## Project Overview

A movie recommendation system built using the **CiaoDVD dataset** and a **vibe coding** approach with Claude as the primary AI assistant. The system implements and compares two recommendation algorithms:

1. **Global Average Baseline** — predicts ratings using the global mean ± user/item bias
2. **User-Based Collaborative Filtering (KNN)** — finds similar users using cosine similarity to predict ratings

---

## Dataset: CiaoDVD

| Property | Value |
|---|---|
| Source | [LibRec Datasets](https://guoguibing.github.io/librec/datasets.html) |
| File | `movie-ratings.txt` |
| Total Ratings | 72,665 |
| Unique Users | ~17,615 |
| Unique Movies | ~16,121 |
| Rating Scale | 1–5 stars (integer) |
| Sparsity | ~99.97% |

The dataset is extremely sparse, meaning most users have rated very few movies — a realistic and challenging recommendation scenario.

## Results

| Model | RMSE | MAE |
|---|---|---|
| Global Average Baseline | 0.9477 | 0.7313 |
| User-Based CF (k=40, cosine) | 1.0767 | 0.7933 |

**Key Finding:** The Global Average Baseline outperforms User-Based CF on this dataset. This is a well-known result with highly sparse datasets — when users share very few co-rated movies, collaborative filtering struggles to find reliable neighbors and the global mean becomes a stronger predictor.
